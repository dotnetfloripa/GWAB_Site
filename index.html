Internet of Things - Controlando e monitorando uma residência com uso de dotNET MicroFramework e Azure

Nesta apresentação falaremos sobre o uso de comunicação de dados entre maquinas (M2M) com o netduino controlando as luzes e o alarme de uma residência. Nossa residência fictícia será controlada e monitorada via web por um serviço que rodará no Windows Azure.

O Objetivo é apresentar de forma clara e prática o que precisamos de tecnologia para termos controle e monitoramento de uma residência. Será também demonstrado o quanto é simples a integração do hardware em nossas vidas por meio das plataformas de desenvolvimento e nuvem da microsoft.

O que será abordado:

  • • Internet of Things
  • • Conhecimentos básicos do Hardware necessário para a apresentação
  • • MicroFramework dotNET
  • • Segurança para a comunicação dos dados de controle
  • • ASP.NET
  • • MQTT
  • • Real-time Web
